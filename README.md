## The core idea

A good HappiHorse or HappyHorse-style workflow should not start with a random prompt.

It should start with a production brief:

```text
Goal -> Input type -> Scene type -> Camera -> Motion -> Lighting -> Style -> Constraints -> Review criteria
```

Example:

```text
Goal: product ad
Input type: image-to-video
Scene type: premium product reveal
Camera: slow push-in from a low angle
Motion: subtle product rotation and light sweep
Lighting: dark studio, soft rim light
Style: realistic technology commercial
Constraints: preserve product shape, no text, no logo distortion, no flicker
Review: product accuracy, smooth motion, no warping, usable 9:16 crop
```

---

## The HappiHorse video prompt formula

Use this structure for text-to-video prompts:

```text
Create a [duration]-second [aspect ratio] video of [main subject] doing [clear action] in [environment].
Camera: [movement, shot type, framing].
Lighting: [lighting direction and mood].
Style: [cinematic, realistic, social, documentary, UGC, commercial, anime, etc.].
Motion: [what moves, how fast, what stays stable].
Audio if supported: [ambient sound, dialogue, music mood, sync requirement].
Restrictions: [no text, no logo distortion, no extra limbs, no flicker, no warped geometry].
```

Example:

```text
Create an 8-second vertical video of a matte black wireless headphone case opening slowly on a reflective dark glass surface. Camera: slow push-in from a low angle, macro product framing. Lighting: soft blue rim light with premium studio reflections. Style: realistic technology commercial, minimal and polished. Motion: case opens smoothly, subtle light sweep crosses the product, background remains stable. Restrictions: no text, no logo distortion, no extra objects, no flicker.
```

---

## Image-to-video prompt formula

Use this when starting from an uploaded source image:

```text
Using the uploaded image as the exact visual reference, create a [duration]-second [aspect ratio] video.
Preserve [product shape, character identity, clothing, logo, color, composition].
Camera: [movement].
Motion: [what should move].
Lighting: [lighting].
Style: [style].
Restrictions: [what must not change].
```

Example:

```text
Using the uploaded product image as the exact visual reference, create a 7-second vertical video ad. Preserve the exact product shape, color, label placement and camera angle. Camera: slow left-to-right dolly with a gentle push-in at the end. Motion: subtle highlight movement across the surface and soft background parallax. Lighting: premium commercial studio lighting. Style: realistic e-commerce product video. Restrictions: no text, no logo distortion, no extra products, no shape changes.
```

## Prompt templates by use case

### Text-to-video template

```text
Create a [duration]-second [aspect ratio] video of [subject] doing [action] in [setting]. Camera: [movement and framing]. Lighting: [lighting]. Style: [visual style]. Motion: [main movement]. Restrictions: [things to avoid].
```

### Image-to-video template

```text
Using the uploaded image as the exact visual reference, create a [duration]-second [aspect ratio] video. Preserve [identity/product/shape/style]. Camera: [movement]. Motion: [motion]. Lighting: [lighting]. Restrictions: [what must not change].
```

### Product video template

```text
Create a [duration]-second product video of [product] on [surface/background]. Camera: [movement]. Lighting: [commercial lighting]. Style: [premium/e-commerce/social ad]. Motion: [product or light movement]. Restrictions: preserve product shape, no fake text, no logo distortion.
```

### UGC ad template

```text
Create a [duration]-second vertical UGC-style video of [creator/person] showing [product or benefit] in [realistic setting]. Camera: handheld but smooth. Lighting: natural. Style: authentic creator content. Motion: [simple action]. Restrictions: no fake text, no distorted hands, no face warping.
```

### Cinematic scene template

```text
Create a [duration]-second cinematic shot of [subject] in [environment]. Camera: [shot type and movement]. Lighting: [cinematic lighting]. Mood: [emotion]. Style: [genre]. Motion: [single clear movement]. Restrictions: no text, no copied franchise elements, no unstable camera.
```

### Social clip template

```text
Create a short [duration]-second vertical clip for [platform] showing [hook subject]. Camera: [dynamic but smooth movement]. Lighting: [lighting]. Style: [social/ad/creator]. Motion: [simple clear motion]. Restrictions: no text, no clutter, no distorted faces, no unstable motion.
```

## Workflow: HappiHorse-style image-to-video production

Image-to-video is often the safer production route because the model has a visual anchor.

### Step 1: Create or upload a strong source image

Good source images have:

- clear subject
- stable identity
- simple composition
- clean edges
- strong lighting
- no fake text
- no confusing hands
- no distorted logos
- enough depth for camera motion

Useful Cliprise pages:

- [AI Image Generator](https://www.cliprise.app/features/ai-image-generator)
- [AI Art Generator](https://www.cliprise.app/features/ai-art-generator)
- [Pro Image Editor](https://www.cliprise.app/features/pro-image-editor)
- [AI Background Remover](https://www.cliprise.app/features/ai-background-remover)

### Step 2: Write motion instructions

Good motion instructions:

- slow push-in
- subtle orbit
- gentle dolly left
- light sweep
- steam rising
- fabric moving
- background parallax
- camera glide
- product rotation
- soft zoom out

Avoid:

- complex hand actions
- sudden jumps
- many moving people
- fast camera shakes
- detailed text animation
- complex physics

### Step 3: Preserve what matters

Use preservation instructions:

```text
Preserve exact product shape, color, label placement and camera angle.
```

```text
Preserve character identity, outfit, facial structure and pose.
```

```text
Preserve original artwork composition, palette and style.
```

### Step 4: Review and iterate

If the output fails:

- reduce motion
- shorten duration
- simplify camera
- use cleaner source image
- add preservation instructions
- test another model
- regenerate only the strongest concept

## Common mistakes

### Mistake 1: Asking for too much

Bad:

```text
Create a 60-second movie trailer with many characters, dialogue, explosions, subtitles and product close-ups.
```

Better:

```text
Create an 8-second cinematic shot of one character walking toward a glowing doorway, slow push-in camera, dramatic rim light, no text, no extra characters.
```

### Mistake 2: Starting with text-to-video when consistency matters

If a product, character or mascot must stay consistent, start with image-to-video.

### Mistake 3: Ignoring camera language

Use camera terms:

- slow push-in
- dolly left
- orbit
- tracking shot
- macro close-up
- wide establishing shot
- locked-off tripod
- handheld but smooth
- crane upward
- low-angle push-in

### Mistake 4: Overusing motion

AI video often works better with one clear motion than five competing motions.

### Mistake 5: Trusting generated text

If text matters, add it later in an editor.

### Mistake 6: Publishing without QA

Always check faces, hands, logos, geometry, flicker and misleading realism.

### Why focus on image-to-video?

Image-to-video is often more controllable because the model starts from a visual reference. This is useful for products, characters, brand mascots, AI art, thumbnails and ad variations.

### What makes a good AI video prompt?

A good prompt includes duration, aspect ratio, subject, action, environment, camera movement, lighting, style, motion and restrictions.

### Should I include audio in the prompt?

Only include audio instructions if the model or workflow supports audio generation, audio input, lip sync or synchronized sound. If not, treat audio as a separate post-production step.
